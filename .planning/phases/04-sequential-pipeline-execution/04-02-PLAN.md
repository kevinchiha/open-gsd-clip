- `async record(entry: Omit<AuditEntry, 'id' | 'timestamp'>): Promise<void>`:
  1. Build full entry with `id: randomUUID()`, `timestamp: new Date().toISOString()`

  3. Parse each line as JSON, cast to AuditEntry

  <name>Task 2: Implement progress-based stale agent detection with hard timeouts</name>
  <files>src/orchestrator/health-monitor.ts, src/orchestrator/health-monitor.test.ts</files>
  <behavior>
    - trackAgent(issueId, runId) starts tracking an agent with startedAt and lastActivityAt set to now
    - recordActivity(issueId) updates lastActivityAt to now
    - untrackAgent(issueId) removes agent from tracking, stops timers when no agents tracked
    - check() detects stale agents (no activity for staleThresholdMs) and calls onStale callback
    - check() detects hard timeout (totalElapsed >= hardTimeoutMs) and calls onStale callback
    - Hard timeout fires before stale threshold if elapsed exceeds both
    - destroy() clears all timers to prevent process hangs
    - Periodic check runs every checkIntervalMs while agents are tracked
  </behavior>
  <action>
Create `src/orchestrator/health-monitor.ts`:

```typescript
import { createChildLogger } from '../shared/logger.js';
import type { HealthConfig } from './types.js';
import { DEFAULT_HEALTH_CONFIG } from './types.js';
```

Define internal `TrackedAgent` interface: `{ issueId: string; runId: string; startedAt: number; lastActivityAt: number }`.

Implement `HealthMonitor` class:
- Constructor: `config: HealthConfig = DEFAULT_HEALTH_CONFIG`, `onStale: (issueId: string, reason: string) => void`
- `private tracked = new Map<string, TrackedAgent>()`
- `private timer: ReturnType<typeof setInterval> | null = null`
- `trackAgent(issueId, runId)`: add to map with `Date.now()` timestamps, call `ensureChecking()`
- `recordActivity(issueId)`: update `lastActivityAt` to `Date.now()`
- `untrackAgent(issueId)`: delete from map, stop timer if map empty
- `private ensureChecking()`: if no timer, start `setInterval(check, config.checkIntervalMs)`
- `private stopChecking()`: clearInterval, set timer to null
- `private check()`: iterate tracked agents, check hard timeout first (totalElapsed >= hardTimeoutMs), then stale threshold (elapsed since lastActivity >= staleThresholdMs). Call onStale with descriptive reason string.
- `destroy()`: clear all tracked agents, stop timer
- `getTrackedCount(): number` -- returns tracked.size (for testing)

Important: On stale/timeout detection, remove the agent from tracking (untrack it) BEFORE calling onStale to prevent double-firing.

Create `src/orchestrator/health-monitor.test.ts`:

Use `vi.useFakeTimers()` for deterministic time control.

Test groups:
1. **trackAgent adds to tracking** -- track an agent, verify getTrackedCount() === 1
2. **recordActivity updates timestamp** -- track, advance time, recordActivity, advance more, verify not stale
3. **stale detection fires callback** -- track agent, advance time past staleThreshold, trigger check, verify onStale called with issueId
4. **hard timeout fires callback** -- track agent, recordActivity periodically but advance past hardTimeout total, verify onStale called
5. **untrackAgent removes from tracking** -- track, untrack, verify getTrackedCount() === 0
6. **destroy clears all state** -- track 3 agents, destroy, verify getTrackedCount() === 0
7. **no double-fire** -- after stale fires, verify same agent doesn't fire again
